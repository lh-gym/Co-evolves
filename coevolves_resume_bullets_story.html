    English talk track

    At Co-evolves, my work sat across the data workflow for a P&C insurance SaaS platform. I helped validate REST API integrations with Postman and Python so carrier and third-party data could be ingested reliably. I then built Python pipelines to automate the ingestion and processing of complex premium datasets from tier-1 carriers. Once the data landed in the warehouse, I wrote dbt SQL validation tests to check completeness, duplicate transactions, premium reconciliation, business rules, and source freshness for mandatory regulatory reporting. The validated data then powered Tableau dashboards for providers to monitor premium volumes and broker retention. On top of the curated data, I ran inferential tests and experimental analysis to identify policyholder demographic factors that influenced renewal probability. So the bullets connect as one flow: reliable data entry, automated processing, trusted validation, client-facing reporting, and analytical insight.
